Common Git Commands

git revert insert commit id number
It reverts an old commit an allows you to change the message of the commit
git reset --soft insert commit id number
allows you to go back to a older commit and make it the most recent one. In addition, you can also undo the the changes by using the soft reset with original recent commit id
git reset --mixed insert commit id number
Is very similar to reset soft, except if you want to undo changes you have to use the git reset HEAD "insert file name" and then do git reset --mixed. To finally revert back to the original commit you had in your log
git reset --hard insert commit id number
changes staging index and working directory to match repository/folder
git rm insert file name
To remove files
git mv previous file name new file name OR
git mv insert file name new folder/file name
changes the name or moves the file into a another repository/folder
  • Rename file in folder
  • git rm previous file name
  • git add new file name
git will be able to tell that you have renamed the file
git checkout insert commit id number -- insert file name
retrieve old versions of a file that has been committed
q
Use this command to quit the git log list
git reset HEAD insert file name
resets files that are in the staging index
git commit --amend -m "Insert message"
Adds modified files to a recent commited log
  • Step1: git checkout insert commit id number -- insert file name
  • Step2: git reset HEAD insert file name
  • Step3: git checkout insert file name
To retrieve an old file in a commit log, and to revert back to the original file
git add .
adds all the modified files to the staging index
git commit -am "insert message"
Adds files that have been modified, and commits them with a message
git clean -n
Checks what unmarked files, git can remove
git clean -f
Removes the unmarked files that hasn't been added in the staging index
ls -la
Lists all the hidden files in the directory/folder
ls
Lists all the files in the directory/folder
insert text editor .gitignore
Using this command will create a gitignore file in the text editor that you have input.In addition, you can specify what you want ignored in the gitignore file
git config --global core.excludesfile ~/.gitignore_global
will ignore all files
mkdir insert new folder name
To make a new folder
touch insert new file name
creates a file that doesn't exist
git ls-tree
list the contents of a tree object. In other words, lists the content within the folder
git help insert git command
Opens up a web page that explains what the command does and the different ways you can use it
git log --oneline
Gives you a one line description of each commit log
git log --since="Year-Month-Date"
shows all the commit log since that date
git log --until="Year-Month-Date"
shows all the commit logs until specified date
git log --author="insert name"
searches the commit log by the authors name
git log --grep="insert any keyword"
searches the commit log that has the keyword/s specified
git log insert commit id number.. insert file name
searches the commit id through the most previous commit log to see if there were any changes to the file name
  1. Step1:git log insert commit id number.. insert file name
  2. Step2:git log -p
See the differences in the file that were you were searching with the commit log
  1. Step1: git log --oneline
  2. Step1: git show insert commit id number
shows the commit message and the git difference in the commit you want to see
  1. Step1: git log --oneline
  2. Step1: git diff insert commit id number
shows the difference between the recent commit to the commit id that you referenced
  1. Step1: git log --oneline
  2. Step1: git diff insert newer commit id number..insert older commit id number
shows the difference between the two commit logs you have entered
git branch
shows the all the branches we have in our local repository and shows the current branch that has been selected
git branch insert new branch name
creates a new branch
git checkout insert different branch name
selects a different branch from the repository
git checkout -b
creates a new branch and switches to it
git diff insert one branch..insert other branch
compares the differences in branches
git diff --color-words insert one branch..insert other branch
compares the differences in branches on oneline
git branch --merged
shows which branches has all the commits to the selected branch
git branch -m old branch name new branch name
changes the name of a branch
git branch -d insert branch name
Deletes a branch
cd ..
goes back a folder
git merge insert other branch name
merges the branches together and changes the files
git merge --abort
Aborts the merge during conflict
git stash save "insert message"
saves a modified state from a branch that you don't want to add in the staging index
git stash list
shows a list of things that were saved in the stash
git stash show insert stash expression
shows what has been changed
git stash show -p insert stash expression
shows more in depth was has been changed
git stash pop insert stash expression
adds files that were in the stash to the current branch and removes the stash it as well
git stash apply insert stash expression
adds files that were in the stash to the current branch
git stash drop insert stash expression
removes the selected stash
git stash clear
clears out all the saved stashes
git remote add alias name insert url
points to the remote that is on the server
git remote
shows all the remotes that we are working with
git remote -v
gives more information about the remotes in the repository
git remote rm insert alias name
removes a remote
git push -u insert alias name insert branch name
pushes the branch up to the alias name and also tracks the branch
git branch -r
will show you the remote branches
git branch -a
will show the remote branches and the other branches
git clone insert url
clones a copy of a project from github to your specified location
git fetch insert alias name
gets the current version of the repository from github
git merge insert alias name /branch name
merging files from remote to current branch
git pull
git fetch + git merge
git push insert alias name :insert branch name
used to remove a remote branch
git push insert alias name insert branch name
pushes branch to up to server
git push insert alias name --delete insert branch name
removes a remote branch
