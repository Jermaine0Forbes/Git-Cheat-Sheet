Common Git Commands

git revert "insert commit id number"
It reverts an old commit an allows you to change the message of the commit
git reset --soft
moves the pointer to another commit
git reset --mixed
changes staging index to match repository/folder
git reset --hard
changes staging index and working directory to match repository/folder
git rm
git mv
changes the name or moves the file into a another repository/folder
git checkout "insert commit id number" -- "insert file name"
retrieve old versions of a file that has been committed
q
Use this command to quit the git log list
git reset HEAD "insert file name"
resets files that are in the staging index
git commit --ammend -m "Insert message"
Adds modified files to a recent commited log
  • Step1: git checkout "insert commit id number" -- "insert file name"
  • Step2: git reset HEAD "insert file name"
  • Step3: git checkout "insert file name"
To retrieve an old file in a commit log, and to revert back to the original file
